## 1. 화면 개요

iPad Notifications는 Lock Screen 및 Notification Center에 표시되는 알림 카드 시스템이다. iPad에서는 iPhone보다 **더 넓은** 알림 카드가 화면 중앙에 표시되며, landscape 모드에서 넓은 화면을 활용한다. iOS 26에서는 **Liquid Glass** 소재가 알림 카드 배경에 적용된다.

이 페이지에서는 두 가지 variant를 다룬다: **Collapsed Stack** (접힌 상태)과 **Expanded** (펼친 상태).

| 항목 | 값 |
|------|-----|
| **디바이스** | iPad Pro 13" (1210 x 834pt landscape) |
| **플랫폼** | iPadOS 26 |
| **표시 위치** | Lock Screen (중앙), Notification Center (상단에서 스와이프) |
| **iOS 26 특징** | Liquid Glass 배경, Stack 애니메이션 개선 |
| **카드 너비** | 600pt (iPad landscape) |
| **Dismiss** | 좌 스와이프 → 삭제/관리, 우 스와이프 → 열기 |

### 4.2 Stack 치수

| 속성 | 값 | 비고 |
|------|-----|------|
| 카드 간 Z-offset | 8pt | 하단 카드가 위로 8pt씩 노출 |
| Scale per depth | 0.96 | 뒤로 갈수록 96%씩 축소 |
| 보이는 카드 수 | 최대 3장 | 4번째+ 완전히 숨김 |
| Stack indicator 텍스트 | "N개의 알림" | 스택 하단에 표시 |
| 탭 동작 | 스택 펼치기 (→ Expanded) | — |
| 스와이프 | 전체 스택 dismiss | — |

## 8. iPad 전용 적응사항

### 8.1 Wide Card Layout

- iPad landscape에서 알림 카드 너비: **600pt** (iPhone: 343pt)
- 화면 중앙 정렬, 좌우 여백 305pt씩
- 넓은 카드에서 본문 미리보기가 더 많이 표시 (3줄까지 가능)

### 8.2 iPad Portrait vs Landscape

| 속성 | Portrait (834 x 1210) | Landscape (1210 x 834) |
|------|----------------------|------------------------|
| 카드 너비 | 500pt | 600pt |
| 좌우 여백 | 167pt | 305pt |
| 본문 미리보기 줄 수 | 2줄 | 3줄 |
| 최대 보이는 스택 | 4-5개 앱 | 3개 앱 (높이 제한) |

### 8.3 Pointer 지원

| 인터랙션 | 동작 |
|---------|------|
| Hover | 카드 미묘한 lift (scale 1.01) |
| Click | 해당 앱으로 이동 |
| Right-click | 관리 옵션 (음소거, 끄기) |
| Swipe (trackpad) | 두 손가락 스와이프로 삭제/관리 |

### 8.4 키보드 단축키

| 단축키 | 동작 |
|--------|------|
| `↑` / `↓` | 알림 간 포커스 이동 |
| `Return` | 포커스된 알림 열기 |
| `Delete` | 포커스된 알림 삭제 |
| `⌘ + Delete` | 모든 알림 지우기 |

### 9.2 Split View

- 각 앱 영역에서 독립적으로 Notification Center 접근 불가
- Notification Center는 항상 전체 화면 overlay
- 배너 알림은 화면 상단 전체 너비에 표시

### 9.3 Stage Manager

- Stage Manager에서도 Notification Center는 전체 화면 overlay
- 배너 알림: 현재 포커스된 윈도우 상단에 표시
- 알림 탭 시 해당 앱의 윈도우가 포커스

## 11. 접근성

| 항목 | 설명 |
|------|------|
| VoiceOver | "알림, [앱이름], [제목], [본문], [시간]" 순서 읽기 |
| Dynamic Type | 제목/본문 크기 조절, 카드 높이 자동 확장 |

## 1. 화면 개요

iPad Home Screen은 앱 아이콘 그리드, 위젯, 그리고 Dock으로 구성된다. iPad는 iPhone보다 더 큰 아이콘 그리드와 Extra Large 위젯을 지원한다. iOS 26에서는 위젯 배경에 **Liquid Glass** 소재가 적용되며, 아이콘에도 미묘한 glass 효과가 추가되었다.

| 항목 | 값 |
|------|-----|
| **디바이스** | iPad Pro 13" (1210 x 834pt landscape) |
| **플랫폼** | iPadOS 26 |
| **아이콘 크기** | 76pt x 76pt |
| **그리드** | 6 x 5 (landscape), 5 x 6 (portrait) |
| **위젯 지원** | Small, Medium, Large, Extra Large |
| **Dock** | 최대 15개 아이콘 + 최근 앱 3개 |

## 4. 아이콘 그리드 치수

### 4.1 앱 아이콘

| 속성 | 값 | 비고 |
|------|-----|------|
| 아이콘 크기 | 76pt x 76pt | `system-ui.md` > App Icon 홈 화면 (iPad) |

| 레이블-아이콘 간격 | 4pt | — |

## 8. iPad 전용 적응사항

### 8.1 Landscape / Portrait 전환

| 속성 | Landscape (1210x834) | Portrait (834x1210) |
|------|---------------------|---------------------|
| 그리드 | 6열 x 5행 | 5열 x 6행 |
| 위젯 배치 | 좌측 또는 상단 영역 | 좌측 또는 상단 영역 |
| Dock 너비 | 더 넓음 (15앱 수용) | 약간 좁음 |
| Extra Large 위젯 | 수평 전체 폭 활용 | 사용 불가 (공간 부족) |

### 8.2 Pointer 지원

| 인터랙션 | 동작 |
|---------|------|
| Hover (아이콘) | 미묘한 scale(1.05) lift 효과 |
| Click | 앱 실행 |
| Long press | Context Menu (Quick Actions) |
| Right-click | Context Menu (Quick Actions) |
| Drag | 아이콘 재배치 모드 진입 |

### 8.3 키보드 단축키

| 단축키 | 동작 |
|--------|------|
| `⌘H` | 홈 화면으로 이동 |
| `⌘Space` | Spotlight 검색 |
| `Fn + H` | 홈 화면 (대체) |

## 9. 멀티태스킹 고려사항

### 9.1 홈 화면과 Split View

- 홈 화면에서 앱 아이콘을 드래그하여 Split View 생성 가능
- Dock에서 앱을 위로 드래그 → Slide Over / Split View

### 9.2 Stage Manager 환경

- Stage Manager 활성 시 홈 화면은 배경으로 표시
- 좌하단에 최근 앱 그룹 표시
- 홈 화면 클릭으로 모든 윈도우 숨기기

## 11. 접근성

| 항목 | 설명 |
|------|------|
| VoiceOver | 아이콘 이름 + 앱 위치 ("행 2, 열 3") 읽기 |
| Dynamic Type | 아이콘 레이블 크기 조절 |
| Reduce Transparency | Dock/위젯 Liquid Glass → 불투명 |
| Reduce Motion | 페이지 전환 animation 제거, 앱 열기 zoom 제거 |
| Bold Text | 아이콘 레이블 bold weight |
| Larger Text | 접근성 설정에서 아이콘 크기 확대 가능 |
